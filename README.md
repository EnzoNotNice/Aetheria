A minimalist, task dashboard designed to simplify focus, structure daily objectives, and streamline your workflow.

---

### Highlights

* **Glassmorphism:** Elegant translucent panels built with high-fidelity blurs and micro-thin outline borders.
* **Dynamic Sidebar:** Instantly filter your active objectives by status (Today, Upcoming, Starred) or Category (Work, Personal, Life).
* **Objective Inspector:** Slide-over detail drawer for editing task titles, priorities, due dates, and extensive markdown notes.
* **Active Clock:** Integrated digital timepiece built directly inside the sidebar panel.
* **Local Sync:** Auto-saves all states to LocalStorage, ensuring your configurations are preserved.

---

### Stack

* React 19
* TypeScript 6.0
* Vite 8.0
* Vanilla CSS3
* Lucide Icons

---

### Quick Start

Install dependencies and run the local development server:

```bash
npm install
npm run dev
```

Compile the optimized static production bundle directly to the `public/` directory:

```bash
npm run build
```
